What changed v0.1 → v0.2

  • OpenClaw + Hermes = control plane (execute + orchestrate under a Local-AI reason layer), not a DMZ.
  • The "air-gap" = default-deny ingestion gates (incl. a human gate for BAA/key-ceremony/compliance/go-no-go).
  • "Data warehouse" = the local-first canonical substrate (Postgres custody + Neo4j/Graphiti graph + MinIO); host is the open call.
  • Canonical vs projection — Notion/AppFlowy/Affine/Plane are all projections, never source of truth.
  • Staged by substitution: Stage 1 Notion → Stage 3 AppFlowy + Affine; Notion ends as a sanitized edge.

Open decisions

  • Canonical-substrate host — Hetzner vs Azure vs hybrid-local.
  • All-cloud vs hybrid-local inference.
  • Azure vs AWS vs mixed shop.
  • JumpCloud go/no-go (Mike → Brian, 48h).
  • DevSecOps/SRE hire (gap).
  • Plane vs Notion/AppFlowy — reconcile team PM surface with projection staging.
  • ERPNext's role vs the canonical graph (source + sanctioned store; PHI separate).

North-star principles

  • Max agent delegation; humans gate only the unavoidable. Default-deny.
  • Own canonical state; everything else is a projection. Migrate by substitution.
  • Adversarial-first; verification is the unit of trust (skeleton + muscle).
